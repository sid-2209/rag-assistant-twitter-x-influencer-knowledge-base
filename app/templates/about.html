About

RAG Assistant for Twitter/X is an AI‑powered workspace for exploring influencer datasets. Upload JSON or CSV files and the system automatically cleans, normalizes, and deduplicates the data through an ETL pipeline, then creates embeddings using OpenAI or free local sentence‑transformers. Questions are answered with retrieval‑augmented generation (RAG), grounding every response in the ingested context.

The service can call any OpenAI‑compatible provider via a custom base URL and model, enabling providers such as Groq and Together AI, as well as your own compatible gateways.

Key features

  • Multipart upload: Seamless dataset ingestion with persistence of the vector store for fast restarts.
  • Dual pipelines: Choose between a lean Vanilla RAG or a LangChain‑based flow (toggle via API).
  • Flexible models: Works with OpenAI, Anthropic, and local LLMs (e.g., LLaMA, Mistral).
  • Lightweight UI: Built with FastAPI and Jinja2—easy to extend into dashboards or a SaaS.
